Mathlib cross-references

crossref-exports

Redirects from external-database identifiers to the corresponding Mathlib documentation. The underlying data lives in crossrefs.json.

Endpoints

  • doc?db=<db>&id=<id> → the declaration's documentation page.
  • src?db=<db>&id=<id> → the declaration's source location.

db is one of wikidata, stacks, or kerodon; id is the identifier in that database. For example, doc?db=wikidata&id=Q83478.
